Geospatial Tool-Box

GeoToolBox (geo-tlbx repository) contains the following tools:

1) code from emLab projects.

2) code for various geospatial libraries (ArcPy, GDAL, Google Earth Engine, GeoWombat).

3) _utils.py files (general, db, vector, raster, geo, gee, plot), which have functions to import into another notebook or python script.

  • For instance, this Jupyter Notebook was used to download and extract information for an ArcGIS Online Web Map of Trail POIs around SB. Point of Interest (POI), road, and stream data were downloaded from OpenStreetMap (overpy) and a digitial elevation model (DEM) was downloaded from USGS 3DEP (py3dep).
















Environmental Markets Lab

Segmenting crop fields across Paraguay (2023-2024)

Programmed a workflow to:
  • train a fully convolutional neural network (from monthly Sentinel-2 / Landsat images and manually digitized crop fields) to predict crop field extents and boundaries,
  • transform semantic segmentations of crop fields to individual instances of crop fields,
  • assess model accuracy, and
  • plot each field's average greenness over time in an interactive web-map.

Monitoring rainwater harvesting technology adoption in the Sahel (2023-2024)

  • Monitored the adoption of demilunes, semicurcular catchment basins, in the Sahel.
  • Classified glacis/degraded land to determine areas suitable for demilune adoption.
  • Presented at the 2023 Geofield Virtual Convening.

Detecting sustainable agriculture practices in Central America (2022)

  • Assessed the ability to detect water-smart agriculture practices, cover crops and crop residue retention, on treatment vs control plots in Central America from 2016-2019 using a time-series of Sentinel-2 derived vegetation indices.
  • Created an interactive notebook for the client to see individual treatment and control plot greenness over time.
